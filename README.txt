This is an example of using MyBatis SQL mapping framework with Spring framework 
for basic CRUD operations, data retrieval with stored procedures, and local 
transactions.

Software and technologies used:
1. Sun Java JRE 1.6.0_25-b06
2. MyBatis 3.0.5
3. MySQL 5.5.15 with 'sakila' sample database
4. MySQL Connector/J 5.1.15
5. SpringSource Tool Suite 2.7.2.RELEASE with the following plug-ins installed:
	 Apache IvyDE
6. MySQL Workbench 5.2.34 CE
